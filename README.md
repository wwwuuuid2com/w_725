<p>Application(“moban-email”)=”email@320588.cn”‘ EMAL 电子邮件<p>
<p>Application(“moban-tel”)=”1888888888″‘ 公司电话<p>
<p>Application(“moban-wangzhi”)=”www.dkewl.com” ‘设置域名<p>
<p>Application(“moban-qq”)=”123456789″ ‘设置你的 QQ 号码<p>
